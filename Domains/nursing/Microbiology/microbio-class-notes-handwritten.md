# Microbiology — Handwritten Class Notes

## Jan 23, 2026 — Observing the Microbial Cell

**Microbial Size Units**
- 1 μm = 10⁻⁶ m → Bacteria are measured in μm
- 1 nm = 10⁻⁹ m → Viruses are measured in nm

**Not all bacteria are microscopic — they have exceptions**

**Bacterial Shapes:**
- Bacilli — rods
- Cocci — spheres
- Spirochetes — spiral form — flexible cell wall
- Spirillum — spiral form — rigid cell wall

**Visible Light Range** = 400–750 nm wavelength
- Depends on contrast between object and its medium
- Wavelength smaller than the object
- Magnification

**Refraction** — is the bending of light as it enters a substance that slows its speed
- Refraction magnifies an image

**Resolution** — limits the ability of how well we can see with a microscope

**Microscope with best external structures → SEM**

**Light microscopy questions:** size, shape, color (gram+/-), arrangement
**Electron microscopy:** internal structures (TEM), external structures (TEM/SEM)

**Refractive Index** — measure of bending of a light ray when passing from one medium to another

## Apr 8, 2026 — Mutations & Genome Structure

**Mutation Types:**

| Type | Mechanism | Result |
|------|-----------|--------|
| **Nonsense** | One base change → Stop codon | No protein |
| **Silent** | One base change → Same amino acid | No change |
| **Missense** | One base change | Altered protein |
| **Frameshift** | Base inserted/deleted | Reading frame shifts → usually nonfunctional |

**Genome Sizes:**

| Organism | Genome Size | Noncoding DNA |
|----------|------------|---------------|
| Prokaryotes | 10⁶–16,000 kb | Less than 15% noncoding |
| Eukaryotes | 2,900–100,000,000 kb | More than 90% noncoding |

**Prokaryote Chromosome:**
- Circular
- Very little noncoding DNA
- Histone-like (Hu, H-NS)
- Nucleoid — no membrane

**Eukaryotic Chromosomes:**
- Lots of noncoding DNA
- Introns + Pseudogenes
- Linear
- True histones

**Plasmids:**
- Usually circular and double-stranded
- Replicate by Rolling Circle Replication
- Can be transferred horizontally between cells
- Can carry antibiotic resistance genes

**DNA Base Pairing:**
- A pairs with T — 2 hydrogen bonds
- G pairs with C — 3 hydrogen bonds
