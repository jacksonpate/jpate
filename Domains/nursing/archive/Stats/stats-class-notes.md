# Statistics — Class Notes

## Apr 13, 2025 — Sampling Distribution

**Sampling Distribution**
- Parameter → Population (large circle)
- Sample size of n → Statistic
- Statistic (from sample)

**Population mean example:**
- μ = (1+2+3)/3 = 6/3 = 2
- Sample pairs: {1,2} → 1.5 | {1,3} → 2 | {2,3} → 2.5

**Possible samples (#5 Pick) without replacement:**
| Sample | x̄ |
|--------|-----|
| 1,1 | 1 |
| 1,2 | 1.5 |
| 1,3 | 2 |
| 2,1 | 1.5 |
| 2,2 | 2 |
| 2,3 | 2.5 |
| 3,1 | 2 |
| 3,2 | 2.5 |
| 3,3 | 3 |

Distribution shape: approaches normal bell curve centered at 2

**Central Limit Theorem**
- Sample size n=4
- S₂ = {3,4,3,1} → x̄₂ = 2.75
- S₃ = {1,1,6,6} → 14/4 = x̄₃ = 3.5

**Sampling Distribution of the Sample Mean**
- 1000(0.087)(0.913) = 79,431 ≥ 10
- √(0.087(0.913)/1000)
- 50(0.7)(0.3) → 10.5 ≥ 10
- √(0.7×0.3)/50
- σ = 0.065
- 0.7 - 2(0.065) = 0.57
- 0.7 + 2(0.065) = 0.83
