---
name: Coverage rule
description: No file gets skipped. Every file must be opened, read, and processed before a phase is marked complete.
type: feedback
---

Do not move on from any folder or section until every single file in it has been opened, read, and processed. No skipping. No assuming a file is irrelevant without reading it first. If a file mentions a person, place, event, or topic that has a page anywhere in the vault — that file gets backlinked to it and vice versa.

Before marking any phase complete, run a full file count. Compare it against how many files you actually touched. If the numbers don't match, go back and find what was missed. Do not sign off until every file is accounted for.

**Why:** Partial passes leave gaps that compound. A file assumed to be irrelevant is often the one with the most context.

**How to apply:**
- At the start of any processing phase, count total files in scope
- Track every file touched
- At end: count again. If delta > 0, go back
- Applies to: Messages/, People/, Journal/, Life/, School/, AI & Tech/, Personal Data/Snapchat/ — every folder
