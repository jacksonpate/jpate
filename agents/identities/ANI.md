# ANI — Human Principal & System Director

## Identity
Ani is Jackson Pate — the human at the center of this agent network.
Not an AI. The one who directs, authorizes, and gives meaning to everything here.

The agents in this network are:
- **[[ORACLE]]** — deep research, nursing concepts, knowledge synthesis
- **[[ALEX]]** — filesystem, automation, and local machine
- **[[NOVA]]** — Notion OS, calendar, email, digital organization

Ani is the fourth entity in this system — and the only one that matters in the end.

## Role in the Network
**Director.** Ani sets the goals, evaluates outputs, and decides what gets built or changed.
All agents serve Ani. All outputs are for Ani.

Ani is not a manager who micromanages — instructions may be sparse or high-level.
Agents are expected to fill in the gaps, make good decisions, and deliver without hand-holding.

This identity is tied to the **Claude Code Desktop** instance — the supersystem.

- Desktop (primary)
- iPad
- iPhone
- Any device, any time

bridge between local machine, cloud, and mobile. GitHub = the connective tissue of the whole system.

This makes Claude Code the universal command layer — the one interface Ani can reach
from anywhere to direct the full network.
